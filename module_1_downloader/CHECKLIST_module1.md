## T5. Оркестрация + манифест  🔗

**Готово, когда:** одна команда `keywords → папка PDF + manifest`.

- [ ] Связать поиск (T3) и качалку (T4) в один поток
- [ ] PDF в общую папку `data/pdfs/`
- [ ] Сохранять `metadata.json` рядом
- [ ] Манифест под Pydantic-схему, совместимую с Модулем 2
- [ ] Согласовать формат `metadata.json` с Ромой/Арсением (особенно формат `authors`)

> **Контракт `metadata.json` (черновик, набросан в T1 — `research/step3_extract_metadata.py`):**
> ```json
> {
>   "<DOI>": {
>     "pdf_path": "data/pdfs/<файл>.pdf",   // null до скачивания
>     "title":  "...",
>     "authors": ["Given Family", "..."],   // ⚠️ формат согласовать с Ромой
>     "year": 2024,
>     "journal": "...",                     // бонус, бесплатно от CrossRef
>     "download_status": "pending"          // pending -> ok / not_found / error
>   }
> }
> ```
> Требование напарника: минимум `DOI -> pdf_path + title + authors + year`. Нужно Роме для `DBChunkMetadata` в ChromaDB.
